# Lua Roguelike RPG - 需求规格说明书

## 1. 项目概述

### 1.1 项目名称
**英雄地牢** (Heroes Dungeon)

### 1.2 项目类型
类CRPG Roguelike 游戏，等距视角，回合制战棋

### 1.3 核心概念
一款结合CRPG角色扮演与Roguelike元素的4人小队地牢探险游戏。玩家选择一个英雄开始游戏，在多层地牢中战斗、成长、解锁新内容。

### 1.4 目标平台
- Windows / Mac / Linux 桌面端
- 移动端 (iOS / Android)

---

## 2. 核心玩法

### 2.1 游戏流程
1. **新游戏** → 选择初始英雄 → 进入地牢第1层
2. **探索地牢** → 战斗 → 获取装备/技能卷轴/资源
3. **死亡与复活** → 单个成员死亡可在当关卡内用技能复活，或在关卡结束后花费资源复活
4. **推进关卡** → 击败当前层Boss → 进入下一层
5. **游戏结束** → 全员死亡 → 结算本次游戏成就
6. **解锁内容** → 达成特定条件 → 解锁新英雄/新掉落物

### 2.2 战斗系统
- **类型**: 回合制战棋 + QTE机制
- **视角**: 等距视角 (Isometric)
- **回合顺序**: 基于速度属性的 initiative 系统
- **范围攻击**: 支持AOE技能（圆形、矩形、扇形）
- **地形利用**: 可利用障碍物、高地等地形优势
- **QTE机制**: 敌人攻击回合时，玩家可触发**弹反**或**闪避**QTE，成功可减少/躲避伤害

#### 2.2.1 QTE战斗机制

| QTE类型 | 触发时机 | 成功效果 | 失败效果 |
|---------|----------|----------|----------|
| **弹反** | 敌人普通攻击 | 反弹部分伤害给敌人 | 受到全额伤害 |
| **闪避** | 敌人攻击前 | 完全躲避攻击 | 受到全额伤害 |

**QTE操作方式**:
- 敌人回合开始时，画面提示QTE指令
- 玩家在限定时间内输入对应按键/滑动
- 根据判定结果计算伤害结算

**判定标准**:
- 完美操作: 额外奖励（反击、暴击等）
- 成功: 正常减免效果
- 失败: 无减免

### 2.3 死亡机制
| 情况 | 处理方式 |
|------|----------|
| 单个角色死亡 | 当关卡内可使用复活技能；关卡结束后花费资源在营地复活 |
| 全员死亡 | 游戏结束，结算本次成就 |

### 2.4 解锁系统
- **完成目标解锁**: 击败特定Boss、通关特定层数、收集特定物品
- **积累解锁**: 累计杀敌数、探索度等
- **解锁内容**: 新英雄角色、新装备配方、新技能卷轴

## 3. 角色系统

### 3.1 英雄设计
游戏包含多个可解锁的英雄，类DOTA英雄设计：

| 英雄类型 | 特点 | 示例 |
|----------|------|------|
| 力量型 | 高生命、高物理伤害 | 坦克、战士 |
| 敏捷型 | 高攻速、闪避、暴击 | 刺客、射手 |
| 智力型 | 高魔法伤害、辅助能力 | 法师、辅助 |

**每个英雄拥有**:
- 独特3D/2D模型（等距视角）
- 初始属性（力量/敏捷/智力）
- 先天被动技能（1-2个）
- 先天主动技能（1-2个）

### 3.2 队伍系统
- **初始**: 选择1个英雄开始游戏
- **招募**: 地牢中可招募NPC英雄加入队伍
- **替换**: 可在营地切换队伍成员
- **最大人数**: 4人

### 3.3 属性系统

#### 3.3.1 核心属性
| 属性 | 英文 | 效果 |
|------|------|------|
| 力量 | STR | +生命上限 +物理伤害 |
| 敏捷 | AGI | +攻击速度 +闪避 +暴击 |
| 智力 | INT | +魔法伤害 +魔法值 +技能效果 |

#### 3.3.2 衍生属性
| 属性 | 效果 |
|------|------|
| 生命值 (HP) | 降至0则死亡 |
| 魔法值 (MP) | 使用技能消耗 |
| 攻击力 | 物理伤害 |
| 防御力 | 减少受到的物理伤害 |
| 魔抗 | 减少受到的魔法伤害 |
| 攻速 | 攻击频率 |
| 暴击率 | 暴击概率 |
| 暴击伤害 | 暴击时伤害倍率 |
| 闪避 | 躲避攻击概率 |
| 速度 | 决定回合顺序 |

### 3.4 技能系统

#### 3.4.1 技能获取
- **先天技能**: 英雄自带（被动/主动）
- **学习技能**: 通过拾取技能卷轴学习

#### 3.4.2 技能类型
| 类型 | 说明 |
|------|------|
| 被动技能 | 常驻生效（如增加属性、概率触发效果） |
| 主动技能 | 消耗MP使用（有冷却时间） |
| 终极技能 | 大招，消耗更高 |

## 4. 地牢系统

### 4.1 地牢结构
- **多层地牢**: 从第1层开始，逐层深入
- **每层主题**: 不同层数有不同主题（哥特、地牢、深渊等）
- **Boss层**: 每5层一个Boss层
- **随机生成**: 每层地图随机生成

### 4.2 关卡类型
游戏包含多种关卡类型：

| 类型 | 说明 | 玩法特点 |
|------|------|----------|
| **常规PVE** | 标准战斗关卡 | 消灭当前房间所有敌人 |
| **Boss战** | 首领战斗 | 单一强大敌人，独特机制 |
| **逃生** | 限时逃离 | 在限定回合/时间内到达出口 |
| **守卫据点** | 防守目标 | 保护指定目标不被敌人摧毁 |
| **解谜** | 谜题关卡 | 利用环境机关、逻辑谜题通关 |

#### 4.2.1 常规PVE
- 标准房间战斗
- 击败所有敌人完成
- 获得战利品

#### 4.2.2 Boss战
- 每5层一个Boss层
- Boss拥有独特技能组和机制
- 击败Boss进入下一层
- 掉落稀有装备

#### 4.2.3 逃生关卡
- 触发条件: 特定事件/层数
- 目标: 在N回合内到达出口
- 限制: 敌人追逐/陷阱触发
- 失败: 队伍伤亡或全灭

#### 4.2.4 守卫据点
- 触发条件: 特定事件/层数
- 目标: 保护NPC/装置N回合
- 失败: 目标被摧毁则失败

#### 4.2.5 解谜关卡
- 触发条件: 特定房间/层数
- 类型: 机关解谜、元素解密、路径谜题等
- 提示: 场景中有线索提示

### 4.3 房间类型
| 类型 | 说明 |
|------|------|
| 普通房 | 小怪战斗 |
| 精英房 | 精英怪战斗 |
| 宝箱房 | 获得装备/道具 |
| 商店房 | 购买装备/道具 |
| 祭奠房 | 花费资源恢复/强化 |
| 事件房 | 随机事件 |
| 楼梯房 | 通往下一层 |

### 4.4 地图元素
- 障碍物（不可通行）
- 可破坏物
- 触发器（陷阱、机关）
- 视线遮挡

## 5. 装备系统

### 5.1 装备类型
| 部位 | 说明 |
|------|------|
| 武器 | 主武器，影响攻击和技能 |
| 防具 | 护甲，影响防御 |
| 饰品 | 戒指/护符，提供特殊效果 |

### 5.2 武器系统（MOBA风格）

#### 5.2.1 合成路线
武器有明确的合成树：
```
基础武器 → 进阶武器 → 最终武器
    ↓           ↓
  变体1      变体2
```

#### 5.2.2 词条机制
- 每个武器有**固定词条**（如"火焰附加"、"吸血"）
- 词条**跟随武器**，不会随机变化
- 词条**数值**可通过消耗资源**升级**

#### 5.2.3 升级系统
- 消耗游戏内资源（金币/材料）升级词条数值
- 升级有成功率机制
- 最高等级限制

### 5.3 掉落机制
- 怪物掉落
- 宝箱掉落
- 商店购买
- 特定掉落物可通过解锁系统解锁获取

## 6. 用户界面

### 6.1 UI风格
**暗黑破坏神风格** - 四周UI面板，中间游戏区域

### 6.2 主界面布局

```
┌─────────────────────────────────────────────┐
│  顶部: 层数信息 / 游戏时间 / 金币            │
├─────────────────────────────────────────────┤
│                                             │
│                                             │
│              游戏视图 (等距)                 │
│                                             │
│                                             │
├─────────────────────────────────────────────┤
│  底部: 队伍面板 (4个角色槽) + 快捷技能栏     │
│        角色HP/MP条  状态图标                │
└─────────────────────────────────────────────┘

左侧面板（可展开）:
- 角色详情
- 背包
- 技能

右侧面板（可展开）:
- 地牢地图
- 任务/目标
- 日志
```

### 6.3 战斗界面
- 显示当前回合角色高亮
- 可选行动按钮（攻击/技能/移动/待机）
- 目标选择辅助（攻击范围、技能范围显示）
- 伤害/治疗数字飘字

### 6.4 菜单系统
- 主菜单: 新游戏 / 继续 / 解锁内容 / 设置 / 退出
- 暂停菜单: 继续 / 保存 / 设置 / 退出
