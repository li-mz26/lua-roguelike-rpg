# Lua Roguelike RPG

一个使用 Lua 开发的回合制 Roguelike RPG 游戏。

## 特性

- 回合制战斗系统
- 随机生成的地牢
- 角色成长系统
- 道具和装备系统

## 快速开始

### 依赖

- [LÖVE 2D](https://love2d.org/) 11.0+ (包含 Lua)

### 运行

```bash
# 克隆仓库
git clone https://github.com/li-mz26/lua-roguelike-rpg.git
cd lua-roguelike-rpg

# 使用 LÖVE 运行游戏
love .
```
