# Lua Roguelike RPG

一个使用 Lua 开发的回合制 Roguelike RPG 游戏。

## 特性

- 回合制战斗系统
- 随机生成的地牢
- 角色成长系统
- 道具和装备系统

## 快速开始

### 依赖

- Lua 5.1+
- [LÖVE](https://love2d.org/) (可选，用于图形界面)

### 运行

```bash
# 克隆仓库
git clone https://github.com/li-mz26/lua-roguelike-rpg.git
cd lua-roguelike-rpg

# 运行游戏
lua main.lua
```
